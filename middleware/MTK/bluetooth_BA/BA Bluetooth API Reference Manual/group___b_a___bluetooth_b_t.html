Overview

This section includes the Bluetooth APIs.

MediaTek MT2523 introduces a platform support for the Bluetooth network stack, which allows a device to wirelessly exchange data with other Bluetooth devices. It provides access to the Bluetooth functionality through the MT2523 Bluetooth APIs. These APIs enable wireless connectivity between Bluetooth devices with point-to-point and multipoint wireless features in various applications.

Classic Bluetooth technology is dedicated for more battery-intensive operations such as streaming and communicating between Bluetooth devices. For Bluetooth devices with low power requirements, MT2523 introduces platform support for Bluetooth Low Energy. To learn more, see Bluetooth Low Energy.
