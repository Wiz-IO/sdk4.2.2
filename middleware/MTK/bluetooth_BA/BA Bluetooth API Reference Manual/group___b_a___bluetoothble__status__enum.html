Enumeration Type Documentation

enum ble_status_t
Deprecated:
This enumeration defines the BLE status codes.
Enumerator
BLE_STATUS_SUCCESS 

The operation completed successfully.

BLE_STATUS_UNKNOWN_HCI_CMD 

The Controller does not understand the HCI Command Packet OpCode sent by the Host.

BLE_STATUS_UNKNOWN_IDENTIFIER 

A command was sent from the Host to identify a connection, but that connection does not exist.

BLE_STATUS_HARDWARE_FAILURE 

It notifies the Host that the Controller operation has failed for an unknown reason and no error code is available.

BLE_STATUS_PAGE_TIMEOUT 

A page timed out because of the Page Timeout configuration parameter.

BLE_STATUS_AUTHENTICATION_FAILURE 

Pairing or authentication failed due to incorrect results in the pairing or authentication procedure.

BLE_STATUS_PIN_OR_KEY_MISSING 

It is used when pairing failed because of a missing PIN, or authentication failed because of a missing Key.

BLE_STATUS_MEMORY_CAPACITY_EXCEEDED 

It notifies the Host that the Controller has run out of memory to store new parameters.

BLE_STATUS_CONNECTION_TIMEOUT 

The link supervision timeout has expired for a given connection.

BLE_STATUS_CONNECTION_LIMIT_EXCEEDED 

An attempt to create another connection failed because the Controller is already at its limit of the number of connections it can support.

BLE_STATUS_SYNCHRONOUS_CONNECTION_LIMIT_TO_A_DEVICE_EXCEEDED 

The Controller has reached the limit of the number of synchronous connections it supports for the device.

BLE_STATUS_ACL_CONNECTION_ALREADY_EXISTS 

An attempt to create a new ACL Connection to a device when there is already an existing connection to this device.

BLE_STATUS_COMMAND_DISALLOWED 

The Controller is unable to process the requested command.

BLE_STATUS_REJECTED_BY_LIMITED_RESOURCES 

An incoming connection was rejected due to limited resources.

BLE_STATUS_REJECTED_BY_SECURITY 

A connection was rejected due to security requirements not being fulfilled, such as authentication or pairing.

BLE_STATUS_REJECTED_BY_UNACCEPTABLE_ADDRESS 

A connection was rejected because this device does not accept the BD_ADDR.

BLE_STATUS_ACCEPT_TIMEOUT 

The Connection Accept Timeout has been exceeded for this connection attempt.

BLE_STATUS_UNSUPPORTED_FEATURE_OR_PARAM 

A feature or parameter value in the HCI command is not supported.

BLE_STATUS_INVALID_HCI_CMD_PARAM 

At least one of the HCI command parameters is invalid.

BLE_STATUS_REMOTE_USER_TERMINATED 

The user on the remote device terminated the connection.

BLE_STATUS_REMOTE_DEVICE_LOW_RESOURCE 

The remote device terminated the connection because of low resources.

BLE_STATUS_REMOTE_DEVICE_POWER_OFF 

The remote device terminated the connection because the device is about to power off.

BLE_STATUS_LOCAL_HOST_TERMINATED 

The local device terminated the connection.

BLE_STATUS_REPEATED_ATTEMPTS 

The Controller forbids an authentication or pairing procedure because too little time has elapsed since the last failed attempt for authentication or pairing.

BLE_STATUS_PAIRING_NOT_ALLOWED 

The device does not allow pairing.

BLE_STATUS_UNKNOWN_LMP_PDU 

The Controller has received an unknown LMP OpCode.

BLE_STATUS_UNSUPPORTED_REMOTE_FEATURE_ORLMP_FEATURE 

The remote device does not support a feature associated with the issued command or LMP PDU.

BLE_STATUS_SCO_OFFSET_REJECTED 

The offset requested in the LMP_SCO_link_req PDU is rejected.

BLE_STATUS_SCO_INTERVAL_REJECTED 

The interval requested in the LMP_SCO_link_req PDU is rejected.

BLE_STATUS_SCO_AIR_MODE_REJECTED 

The air mode requested in the LMP_SCO_link_req PDU is rejected.

BLE_STATUS_INVALID_LMP_OR_LL_PARAM 

Some LMP PDU / LL Control PDU parameters are invalid.

BLE_STATUS_UNSPECIFIED_ERROR 

No other error code specified is appropriate to use.

BLE_STATUS_UNSUPPORTED_LMP_OR_LL_PARAM_VALUE 

An LMP PDU or an LL Control PDU contains at least one parameter value that is not supported by the Controller at this time.

BLE_STATUS_ROLE_CHANGE_NOT_ALLOWED 

A Controller will not allow a role change at this time.

BLE_STATUS_LMP_OR_LL_RESPONSE_TIMEOUT 

An LMP transaction failed to respond within the LMP response timeout or an LL transaction failed to respond within the LL response timeout.

BLE_STATUS_LMP_ERROR_TRANSACTION_COLLISION 

An LMP transaction has collided with the same transaction that is already in progress.

BLE_STATUS_LMP_PDU_NOT_ALLOWED 

A Controller sent an LMP PDU with an OpCode that was not allowed.

BLE_STATUS_ENCRYPTION_MODE_NOT_ACCEPTABLE 

The requested encryption mode is not acceptable at this time.

BLE_STATUS_LINK_KEY_CANNOT_BE_CHANGED 

Link key cannot be changed because a fixed unit key is being used.

BLE_STATUS_REQUESTED_QOS_NOT_SUPPORTED 

The requested Quality of Service is not supported.

BLE_STATUS_INSTANT_PASSED 

An LMP PDU or LL PDU that includes an instant cannot be performed because the instant when this would have occurred has passed.

BLE_STATUS_PAIRING_WITH_UNIT_KEY_NOT_SUPPORTED 

Impossible to establish pairing as the requested unit key is not supported.

BLE_STATUS_DIFFERENT_TRANSACTION_COLLISION 

An initiated LMP transaction collided with an ongoing transaction.

BLE_STATUS_QOS_UNACCEPTABLE_PARAM 

The specified Quality of Service parameters could not be accepted.

BLE_STATUS_QOS_REJECTED 

The specified Quality of Service parameters cannot be accepted and Quality of Service negotiation should be terminated.

BLE_STATUS_CHANNEL_CLASSIFICATION_NOT_SUPPORTED 

The Controller cannot perform channel assessment because it is not supported.

BLE_STATUS_INSUFFICIENT_SECURITY 

Sending the HCI command or LMP PDU is only possible on an encrypted link.

BLE_STATUS_PARAM_OUT_OF_RANGE 

A parameter value requested is outside the mandatory range of parameters for the given HCI command or LMP PDU and the recipient does not accept that value.

BLE_STATUS_ROLE_SWITCH_PENDING 

A Role Switch is pending.

BLE_STATUS_RESERVED_SLOT_VIOLATION 

The current Synchronous negotiation was terminated with the negotiation state set to Reserved Slot Violation.

BLE_STATUS_ROLE_SWITCH_FAILED 

A role switch was attempted but it failed and the original piconet structure is restored.

BLE_STATUS_EXTENDED_INQUIRY_RESPONSE_TOO_LARGE 

The extended inquiry response with the requested requirements for FEC is too large to fit in any of the packet types supported by the Controller.

BLE_STATUS_SSP_NOT_SUPPORTED_BY_HOST 

The IO capabilities request or response was rejected because the sending Host does not support Secure Simple Pairing even if the receiving Link Manager does.

BLE_STATUS_HOST_PAIRING_BUSY 

The Host is busy with another pairing operation and unable to support the requested pairing.

BLE_STATUS_NO_SUITABLE_CHANNEL_FOUND 

The Controller could not calculate an appropriate value for the Channel selection operation.

BLE_STATUS_CONTROLLER_BUSY 

The operation was rejected because the Controller was busy and unable to process the request.

BLE_STATUS_UNACCEPTABLE_CONNECTION_PARAM 

The remote device either terminated the connection or rejected a request because of one or more unacceptable connection parameters.

BLE_STATUS_DIRECTED_ADVERTSING_TIMEOUT 

Directed advertising completed without a connection being created.

BLE_STATUS_MIC_FAILURE 

The connection was terminated because the Message Integrity Check (MIC) failed on a received packet.

BLE_STATUS_CONNECTION_FAILED_TO_BE_ESTABLISHED 

The LL initiated a connection but the connection has failed to be established.

BLE_STATUS_MAC_CONNECTION_FAILED 

The MAC of the 802.11 AMP was requested to connect to a peer, but the connection failed.

BLE_STATUS_COARSE_CLOCK_ADJUSTMENT_REJECTED 

The master is unable to make a coarse adjustment to the piconet clock at a moment using the supplied parameters.

The base of the HCI status.
