Detailed Description

Deprecated:
This event (BLE_SM_RETRIEVE_KEYS_REQ) structure enables getting the keys stored in the flash memory, such as file system.

If remote_address is available, the keys are stored for each device, since they are different for each remote device. If remote_address is not available, the keys are only for the local device, such as local IRK/CSRK/counter. Generate a unique local IRK/CSRK and set initial counter to 0, which will be retrieved by the SM. The SM updates the local counter with the new user provided counter.
